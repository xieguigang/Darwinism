run {docker} R Documentation

Run a command in a new container.(这个函数会捕捉到命令的标准输出然后以字符串的形式返回)

Description

Usage

run(container, command,
    script = NULL,
    workdir = NULL,
    mounts = NULL,
    portForward = NULL,
    ... = NULL,
    shell.cmdl = FALSE);

Arguments

command

[as string]

args

the commandline arguments for the shell `command` run inside a docker container. [as list]

shell.cmdl

this debug parameter specific that just returns the commandline for run docker instead of run command and returns the std_output. [as boolean]

Details

这个方法能够自定义的参数比较有限,如果需要更加复杂的使用方法,可以使用Environment对象

this function returns data object of type string.
