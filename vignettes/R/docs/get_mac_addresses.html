Retrieve MAC Addresses of Network Interfaces Scans system network interfaces and extracts MAC addresses, excluding loopback interfaces.

Description

Usage

get_mac_addresses();

Arguments

Details

Authors

xieguigang

Value

A named list where: \itemize{ \item Names are network interface names (e.g., "eth0") \item Values are corresponding MAC addresses (empty string if unavailable) }
